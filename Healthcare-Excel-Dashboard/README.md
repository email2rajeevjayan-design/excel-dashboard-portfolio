# Healthcare Excel Dashboard

## Project Overview

This dashboard analyzes healthcare operational data using Excel to monitor patient admissions, department performance, and operational KPIs.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Conditional Formatting

## Key KPIs

- Total Patients
- Admissions
- Department Performance
- Average Stay
